University Bootcamp Project (2021)

Alone is a 2D platformer game, which times you as you play the game and the goal is to finish all the levels in the shortest time possible! My record is 73 seconds, so slow, so I'm hoping you try to beat me ;)

I used classes in order to be able turn each platform into its own object which could be edited and used in loops in order to, for example, calculate collisions.

This was my first project in Java, and I am proud of what I achieved from just 2 weeks of learning a new programming language, although now I can see ways in which I would have rewritten certain parts of my code to improve readablitiy and efficiency, which I will use to further progress as a computer scientist.
